# Blog / CMS

Content-driven blog with MDX support, categories, search, RSS feed, and syntax highlighting. Built with Next.js and @next/mdx.

**Stack:** Nextjs | **Complexity:** moderate
**Tags:** `blog`, `cms`, `mdx`, `markdown`, `content`, `rss`, `syntax-highlighting`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```

## Claude Code

This project includes a `CLAUDE.md` file optimized for development with
[Claude Code](https://claude.ai/code). Open the project and Claude will
understand the architecture, commands, and conventions automatically.
